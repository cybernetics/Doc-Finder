Ειδικό λεξικό:
	internet: Ένα παγκόσμιο δίκτυο υπολογιστών
	server: Ενας υπολογιστής ή πρόγραμμα το οποίο διαχειρίζεται κάποια πληροφορίες/υπηρεσίες ενός δικτύου.
	client: Ενας υπολογιστής ή πρόγραμμα το οποίο ειναι ζητάει και μετά δέχεται πληροφορίες/υπηρεσίες απο έναν server.
	GET: Είναι μια μέθοδος του προτοκόλου HTTP με τον οποίο μπορούν να μεταφερούν πληροφορίες απο τον server στον client.
	web browser engine: Είναι ενα πρόγραμμα το οποίο δέχεται αρχεία δεδομένων, εικόνων κτλπ και πληροφορίες μορφοποίησης, παρουσίασης και εμφανίζει το αποτέλεσμα στην οθόνη.
	webkit, Gecko, Trident ή Presto: Web Browser Engines πάνω στους οποίους είναι βασισμένοι οι διμοφιλέστεροι φυλλομετρητές του internet.
	Javascript: Γλώσσα μορφοποίησης δεδομένων που τρέχει στον υπολογιστή του client.
	Javascript Framework: Ένα σετ απο βιβλιοθήκες βασισμένες στην Javascript.
	web console: Προγραμμα που παρέχουν ολοι οι μοντέρνοι φυλλομετρητές για να βοηθήσει τους developers στην παραγωγή λειτουργικού κώδικα.

Απαίτησεις:
	Απαίτησεις Συστήματος:
			Μια ενεργή σύνδεση στο internet με την οποία θα μεταφέρονται τα δεδομένα απο τον server στον client (φυλλομετρητής) με την χρήση της μεθόδου "GET".
			Ένας μοντέρνος φυλλομετρητής βασισμένος σε webkit, Gecko, Trident ή Presto web browser engine.
			Ο φυλλομετρητής πρεπει να έχει την δυνατότητα εκτέλεσης Javascript κώδικα βασισμένο στο "Javascript Framework" με όνομα "JQuery".

	Απαίτησεις Χρήστη:
		Μια ενεργή σύνδεση στο internet.
		Ένας μοντέρνος φυλλομετρητής.


Σενάρια Χρήσης:
		Ο χρήστης πρεπει:
			*να συμπληρώσει το keyword για το οποίο θέλει να κάνει την αναζήτηση στο textfield
			*και μετα πρέπει να επιλέξει αν θελει να ψάξει για κάποια γλώσσα προγραμματισμού συγκεκριμένα ή να ψάξει για ολες απο το dropdown menu
			*τέλος να εκτελέσει την αναζήτηση κάνοντας submit την φόρμα είτε πατόντας <enter> είτε μέσω του "submit" κουμπιού της φόρμας.
		Με εκτελεστεί η εντολή λήψης τον δεδομένων απο τον server:
			*Αν ο server έχει επιστρέψει επιτυχώς τα δεδομένα θα πρέπει τα δεδομένα αυτά να διαμορφοθούν κατάλληλα και μετά να συμπληρωθούν στην HTML φόρμα εξόδου (div με ID = results και info)
			*Αλλιώς να επιστρέψει error μήνυμα που να εξηγεί πού ειναι το πρόβλημα μέσω του web console του φυλλομετρητής.
		Αφού εχουν εμφανιστεί τα δεδομένα στην οθόνη και αν τα αποτελεσματά που βρέθηκαν ειναι πάνω απο 10 τοτε ο χρήστης έχει την δυνατότητα να μεταφερθεί σε άλλες σελίδες για να βρεί την πληροφορία που ψάχνει.
